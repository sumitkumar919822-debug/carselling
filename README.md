# SellerCar Marketplace

SellerCar is a static web application built with HTML, CSS, and JavaScript that lets users browse and sell vehicles. The app supports cars, bikes, and cycles, with a landing page, filtered marketplace, cart, and listing creation form.

## Features

- Landing page with app introduction and animated background slideshow
- Category filtering for cars, bikes, and cycles
- Search by title, brand, or description
- Sorting options for price and brand
- Product cards with real vehicle images
- Cart sidebar with live item count and total price
- Add new listings through a form
- Responsive layout for desktop and mobile

## Files

- `index.html` - Main app markup and landing page structure
- `styles.css` - Styling for landing page, marketplace, cards, and forms
- `script.js` - App logic for filtering, cart operations, landing navigation, and background slideshow
- `assets/images/` - Local image assets and illustrations for the landing page

## Getting Started

1. Open `index.html` in a web browser.
2. Use the landing page buttons to enter the application or learn more.
3. Browse listings and use filters or sort options to find vehicles.
4. Add items to the cart and review the total.
5. Use the "Add Listing" form to create a new vehicle listing.

## Notes

- This is a static front-end application and does not store data permanently.
- Images are loaded from external URLs for the product cards.
- The landing page background changes automatically every 3 seconds.
